// What We Built

AI that knows its
boundaries.

As AI systems grow more autonomous, the question isn't just what they can do — it's what they're allowed to do. SAACGAIS is a proof-of-concept healthcare platform that layers two specialized AI agents on top of a secured backend, each operating only within its approved scope.

Clinicians get AI-powered summaries and health suggestions. Administrators control access. Patients stay protected. Every AI call passes through an authorization checkpoint before touching sensitive data.

// Key Features

Built for security.
Designed for clarity.

Five capabilities that make SAACGAIS a meaningful exploration of secure multi-agent AI in practice.

🤖
Dual AI Agent System

Two AWS-hosted AI agents with distinct, non-overlapping roles: one for clinical summarization and one for patient health suggestions. Neither agent can act outside its defined context.

🔐
OPA Authorization Layer

Every request that touches sensitive data — including AI agent calls — is validated against Open Policy Agent rules before execution. Access is denied by default; explicitly allowed by policy.

🪪
Role-Based Access Control

Patients, Clinicians, and Administrators each operate within a strict permission matrix. Roles are verified via JWT tokens issued by AWS Cognito on every request.

🏥
Healthcare-Focused UI

A full-stack Kotlin + Node.js web application tailored for clinical workflows: provider selection, patient records, file uploads, and an AI chat interface for real-time suggestions.

📋
Formal AI Context Documents

Each agent's role, data access scope, and behavioral constraints are defined in an AI Contexts document — a blueprint for responsible deployment of agents in sensitive domains.

// System Architecture

How the pieces fit together.

Authentication is separated from authorization. Cognito handles identity; OPA handles what that identity is allowed to do. Agents sit behind both layers.

// Getting Started

Run it locally.

Prerequisites: Node.js v18+, npm, and AWS credentials configured in your environment.

terminal
# 1. Clone the repo
git clone https://github.com/
    SAACGAIS-Team/SAACGAIS
 
# 2. Start the frontend
cd app/client
npm install
npm start
 
# 3. Start the backend
cd ../server
npm start
// Prerequisites
▸ Node.js v18+ — required for both client and server
▸ AWS Account — Cognito user pool and Bedrock access needed for AI agents
▸ OPA Server — run locally or via Docker for authorization policy enforcement
▸ Environment Variables — configure .env with your AWS credentials and Cognito pool IDs
